# Smile and Aesthetic Dental Care Website

This is the source code for the **Smile and Aesthetic Dental Care** website, built for Dr. Niveditha Bhasi (Kerala, India) using React.

## Features

- Modern responsive design with a custom theme and elegant Google Fonts
- Main navigation and smooth scrolling across sections
- Appointment request form with treatment and contact details
- Google Maps link for easy location access
- Social media links (Instagram, Facebook)
- Beautiful, custom animated cursor

## Getting Started

To run the project locally:

1. **Clone the repo**

   ```
   git clone https://github.com/shubhamx939/smile-aesthetic-dental.git
   cd smile-aesthetic-dental
   ```

2. **Install dependencies**

   ```
   npm install
   ```

3. **Start the development server**

   ```
   npm start
   ```

   Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

## Build and Deployment

- **To build for production:**

  ```
  npm run build
  ```

- **To deploy (used for GitHub Pages):**

  ```
  npm run deploy
  ```
